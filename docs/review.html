🎯 The five that mattered most

  1. Compare unit price, not line total — a 6-pack line vs a single would break every verdict.
  2. Keep item identity clean — fuzzy-match + a "is this the same item?" prompt + a merge tool.
  3. Never double-import a receipt — fingerprint each import and warn on repeats.
  4. Throttle the PIN — lockout after repeated fails so a public URL can't be brute-forced.
  5. Let you export your data — one-tap download so your history is never trapped or lost.

Every finding above was surfaced and judged before approval — this is the "try to break it, then fix it" step, made visible.
